# User-flow guide

1. Sign in (or demo-local) → Command centre
2. Create trip (Setup / Vault / Templates)
3. Add travellers
4. Search mock flights/hotels via gateway → save to trip/itinerary
5. Use Travel services for car/transfer/cruise/activities (planning)
6. AI Planning / Concierge → generate guidance → convert to itinerary (locked items preserved)
7. Budget + packing + documents + reminders
8. Share with collaborator → they accept → edit/view per role → revoke
9. Export ICS / print itinerary / export account data
10. Offline indicator shows when network drops; sync retries when online
